HTML Basics

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures the content and tells the browser what to display.

Key Elements:

  • <!DOCTYPE html> – Declares the document type (HTML5).
  • <html> – The root element.
  • <head> – Contains meta-information, like the page title and links to CSS files.
  • <body> – Holds the visible content.

Example Structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a basic HTML page.</p>
  </body>
</html>

HTML Forms

Forms allow you to collect user input through various elements:

  • <form> – The container for form elements.
  • <input> – For text, email, passwords, etc.
  • <textarea> – For multi-line text input.
  • <button> – For form submission or actions.
  • <select> and <option> – For dropdown menus.

Simple Form Example:

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Submit</button>
</form>

Semantic HTML

Semantic elements provide meaning to your content, improving accessibility and SEO.

Common Semantic Elements:

  • <header> – Contains introductory content or navigation.
  • <nav> – Contains navigational links.
  • <main> – The primary content of a document.
  • <article> – A self-contained piece of content.
  • <footer> – Contains footer information.

Using semantic HTML helps browsers and assistive technologies understand your content structure better.
