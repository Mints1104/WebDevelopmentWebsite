CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. Its syntax consists of a selector and a declaration block.

For example:

selector {
  property: value;
}

This tells the browser which elements to style and how.

Additional example:

p {
  color: #333;
  font-size: 16px;
  line-height: 1.5;
}

Layout & Box Model

The box model represents every element as a box with:

  • Content: The actual content.
  • Padding: Space between content and border.
  • Border: The edge around the padding.
  • Margin: Space outside the border.

For example:

div {
  width: 200px;
  padding: 20px;
  border: 5px solid #000;
  margin: 10px;
}

Also, setting box-sizing: border-box; ensures that padding and border are included within the set width:

div {
  width: 200px;
  padding: 20px;
  border: 5px solid #000;
  box-sizing: border-box;
}

Transitions & Animations

Transitions allow smooth changes between property values. For instance:

button {
  transition: background 0.3s ease;
}

This changes the background color smoothly when the button is hovered.

Animations use keyframes to define style changes over time. For example:

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

You can apply this to elements to animate them on load:

.element {
  animation: fadeIn 0.5s ease-in-out;
}
